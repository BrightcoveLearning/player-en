Use in HTML

If you are using the URLs for the plugins directly in an HTML page, you would use a <script> tag for the JavaScript and a <link> tag for the CSS. Examples follow:

<script src="https://players.brightcove.net/videojs-vr/1/videojs-vr.min.js"></script>
<link href="https://players.brightcove.net/videojs-vr/1/videojs-vr.css" rel="stylesheet">
