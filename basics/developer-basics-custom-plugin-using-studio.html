Player Developer Basics: Custom Plugin - Using Studio

In this topic, you will learn how to assign a custom plugin to a Brightcove Player using Studio.
 

Steps

  1. In Studio, select the PLAYERS module.
  2. Click on the player to which you want to add the custom plugin.
  3. In the player properties, on the left-hand side select the Plugins option .
  4. From the Add a Plugin dropdown, select Custom Plugin.
  5. Add the following:
    • Plugin Name
    • JavaScript URL
    • CSS URL - if needed
    • Options(JSON) - if needed
  6. Save the player.
  7. Publish the player.
