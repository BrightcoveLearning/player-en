Introduction

Bumper videos play before the content video in the player. The poster is displayed for the content video, but when the player starts to play the bumper is shown first followed by the content video. In the following example a demo ad is shown followed by the content video.

The content video's poster, title and short description are displayed, but when play initiates the bumper is played then the content video.

Implementation

There are three ways to implement a video bumper, the are:

  • Using an attribute in the advanced (embed) code
  • Using a query string in the standard (iframe) code
  • Adding to the player's JSON configuration
